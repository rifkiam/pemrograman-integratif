# Implementasi CRUD GRPC dan protobuf dengan NODE JS
## package yang digunakan:
* node.js
* mysql

## tata cara pemakaian:
  - masuk ke direktori
  - jalankan ```npm install```
  - jalankan ```node server.js```
  - testing dapat dilakukan via postman sbb
